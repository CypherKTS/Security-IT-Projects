# Cryptographic Hash Functions - SHA-256 & HMAC Implementation & Python Calculator in Visual Studio Code

This project demonstrates the use of SHA-256, HMAC, Python libraries, and VS Code for message hashing and authentication.

## Features:
- **SHA-256 Hashing**: Computes the hash of a given message.
- **HMAC**: Verifies message authenticity using a shared secret key.

## Tools & Libraries:
- Python `hashlib` for SHA-256.
- Python `hmac` for HMAC.

## What I Learned:
- **HMAC**: How to authenticate messages using a secret key.
- **SHA-256**: Basics of hashing and its application in data security.
